# Guess My Number 🎲

A simple JavaScript guessing game where the player tries to find the secret number between 1 and 20.

## About the Project

This project was created to practice JavaScript fundamentals, DOM manipulation, events, conditions, score updates, and user feedback.

## Technologies Used

- HTML
- CSS
- JavaScript
- DOM Manipulation

## Features

- Random secret number generation
- Score system
- High score tracking
- Input validation from 1 to 20
- Error message when the user does not enter a number
- Visual feedback with background color changes
- Reset button to start a new game

## How to Play

1. Enter a number between 1 and 20.
2. Click the **Check** button.
3. The game will tell you if your guess is too high or too low.
4. Keep guessing until you find the correct number.
5. Click **Again** to restart the game.
