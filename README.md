# Guess My Number 🎲

A simple JavaScript game where the player tries to guess a secret number between 1 and 20.

## Technologies
- HTML
- CSS
- JavaScript
- DOM Manipulation

## Features
- Random secret number
- Score system
- High score
- Check button
- Again button
- DOM updates with JavaScript
